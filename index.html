Java Email Generator

This is a basic java project applying on Object-Oriented design.
Creating automated email using firstname, lastname and department using user input.
Creating a system generated password assigning to the associate email.
Giving option to the user to Change Password
Giving option to Change Mailbox Capacity
Giving option to Set Alternate Email
Store data in file and show the file

Tools used

Intellij IDE || JDK -17 || Git Bash
